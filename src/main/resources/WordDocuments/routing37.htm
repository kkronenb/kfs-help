Routing

•      After initiation the Effort Certification Recreate document takes on the characteristics of a regular effort certification report and routes as normal with this exception:

Recreate Workgroup: After completing the normal effort certification report routing, the document routes to the KFS-SYS Contracts and Grants Manager role for approval. In the default configuration, these users are permitted to modify effort reports. This ability can be turned off by removing the KFS-EC Use Transactional Document permission from this role. If these users modify effort, an FYI action request will be sent to the fiscal officer(s) of the modified account(s).
