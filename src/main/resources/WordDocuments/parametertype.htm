Parameter Type

>>>>>

The parameter type broadly defines what a parameter is used for. In the KFS base data there are only two parameter types: parameters used for document validation  (such as business rules) and parameters used for System Configuration (customizing KFS for your institution).

The Parameter Type document allows you to create new or edit existing parameter types. Code must be developed to support any additional types or changes to existing types.

Only members of the KR-SYS Technical Administrator or KFS-SYS Manager role can create or edit Parameter Type documents and they do not route for approval.
