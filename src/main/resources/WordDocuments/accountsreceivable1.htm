Accounts Receivable

Five custom document searches are available for Accounts Receivable.

Table 3   Accounts Receivable submenu group

Custom Search

Description

Customer Invoices

Allows you to search for invoices.

Customer Credit Memos

Allows you to search for customer credit memos.

Customer Invoice Writeoffs

Allows you to search for writeoffs that have been processed.

Cash Controls

Allows you to search for funds in the Accounts Receivable module, applied or unapplied.

Payment Applications

Allows you to search for payment application documents that have been processed.

 

For information about basic document search, see Using Doc Search to Find a Document in the Overview of the Kuali Financial System.
