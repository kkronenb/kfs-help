Database/Using Criteria Corrections Method  

1.     Select Database and Using Criteria and click  in the Correction Process tab.

The available files for editing appear in the Origin Entry Group list in the Documents in System tab.

2.     Select the file in the Documents in System list and click .

3.     Enter search and modification criteria in the Edit Criteria tab. Click  as you add each criterion.

4.     Select the type of correction file to be created in the Edit Options and Action tab.

5.     Click  to process your LLCP document.

A new file is created for addition to the G/L processing cycle and a message indicates that the document was successfully submitted.

6.     Review the Summary tab.
