Figure 23 - Federal Function document

The Federal Function document includes the Edit Federal Function tab. The system automatically enters data into both the Old and New sections. Selected data fields are available for editing.

Table 11    Edit Federal Function tab definition

Title

Description

Federal Function Code

A code that identifies a specific federal function that will be used as an attribute of higher education function code for federal reporting

Federal Function Name

Required. A descriptive label of this specific federal function code

Active Indicator

 

Indicates whether this federal function code is active or inactive. Remove the check mark to deactivate a federal function code.
