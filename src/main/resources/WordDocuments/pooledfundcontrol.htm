Pooled Fund Control 

 >  >  >

The Pooled Fund Control Maintenance document allows you to create and change pooled funds for internal investment of endowments. For each pooled fund, you may set up specific criteria for the valuation and generation of control offsets and General Ledger values regarding daily activity. The fields in each Pooled Fund record instruct the system regarding the criteria for automated transaction generation and maintenance of the pool control record.
