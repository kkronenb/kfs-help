Edit Option

When you select the File Upload/Manual Edit Correction method, the system displays the edit button below the loaded data.

Clicking edit changes the tab to handle a manual edit.

1.     Click the  button corresponding to the row you want to edit (see below). Then make changes in the manual edit section with editable boxes below the loaded rows.

The loaded row is now available for editing:

2.     Click  when you finish editing the row. The changes you made are immediately reflected in the loaded row.
