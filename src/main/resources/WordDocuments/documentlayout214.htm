Carrier document

The Carrier document includes the Edit Carrier tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Edit Carrier tab definition

Title

Description

Carrier Code

A unique code to identify a particular carrier.

Carrier Description

Required. A description or name of the carrier.

Active Indicator

 

Indicates whether this carrier code is active or inactive. Remove the check mark to deactivate the code.
