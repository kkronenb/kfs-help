Document Layout

The Asset Retirement Reason document includes the Edit Asset Retirement Reason tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Edit Asset Retirement Reason tab definition

Title

Description

Retirement Reason Code

A unique code to identify a retirement reason.

Retirement Reason Name

Required. A descriptive name for the retirement reason.

Retirement Reason Restriction Indicator

Optional. Indicates if the use of this retirement reason is to be restricted. Clear the check box if it is not restricted.
