Making a Balance Inquiry

To access any of the balance inquiries and perform a search, follow these steps:

1   Select the desired inquiry from the Balance Inquiries menu group on the Main Menu tab.

The system displays the requested lookup screen. This screen may display default values; these values may be changed.

 

2   Enter values in all fields preceded by an asterisk. Narrow the search as desired by including additional criteria

3   Click search when you have entered all necessary criteria and are ready to retrieve the balances.
