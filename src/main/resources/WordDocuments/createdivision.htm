Create Division

The Vendor document offers a unique option from the Vendor Detail Lookup screen, called Create Division.

The Create Division option allows you to group entities under the same corporate office with the same Tax ID ('parent vendor'). You might use this feature to add information about the separate divisions or branches, without having to duplicate the corporate information. A division has a different name from the parent.

Parent - Child Vendor Number

The base vendor number is always followed by a hyphen, followed by the sequence number. The sequence number for the Corporate Vendor Number is 0 whereas the Division Vendor Numbers is a 1 or greater. For example, the vendor number for the Cornell University is 1001-0 ('parent') whereas the Cornell Medical School is 1001-1 ('child') in the example shown below.

The create division link is available only for the 'parent' vendors. When you click the create division link, a new Vendor document is initiated and the information from the Corporate Information section of the Vendor tab is copied from its Corporate Vendor and remains display-only.
