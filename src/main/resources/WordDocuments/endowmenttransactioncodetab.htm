Endowment Transaction Code Tab

The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Endowment Transaction Code tab definition

Title

Description

Transaction Code

Required when initiating a document; otherwise, display only. Also known as Etran Code. The code that uniquely identifies system processing (including generation of appropriate General Ledger entries) required for a specific purpose. For example, your institution may set up different Etran codes to specify different handling for transactions involving petty cash, student loans, mutual funds, etc.

Transaction Code Description

Required. Your institution’s description of this Etran code (Petty Cash, Student Loans, Mutual Funds, etc.).

Transaction Code Type

Required. The accounting code type associated with this Etran code. Values are ‘Asset,’ ‘Liability,’ ‘Income,’ and ‘Expense.’ Select from the list or use the lookup  to find the appropriate type.

  Each object code specified in the GL Links tab must be of the type specified in this field. For information on the GL Links tab, see GL Links Tab.

Corpus Indicator

An indicator specifying whether transactions using this Etran code affect the total corpus balance of a true endowment. Check the box to indicate that the transactions affect corpus.

Active Indicator

 

Indicates whether this Etran code is active or inactive. Remove the check mark to deactivate this Etran code.
