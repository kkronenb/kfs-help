Tickler Type Code Maintenance document

The Tickler Type Code Maintenance document includes the Edit Tickler Type Code tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Edit Tickler Type Code tab definition

Title

Description

Tickler Type Code

Required when initiating a document; otherwise, display only. A reason a person or department may want to be notified of upcoming events or actions to be taken.

Tickler Type Code Description

Required. Your institution’s description of this tickler type code; clarifies the general purpose of the tickler.

Active Indicator

 

Indicates whether this tickler type code is active or inactive. Remove the check mark to deactivate this Tickler Type Code record.
