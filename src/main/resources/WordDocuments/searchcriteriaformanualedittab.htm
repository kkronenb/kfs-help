Search Criteria for Manual Edit Tab

When manually editing a file, you can use the Search Criteria for Manual Edit tab to apply a filter to the rows of data and Display-only the rows that match your search criteria. This approach can be helpful when working with a large file that needs corrections to only a limited number of rows. To filter the rows displayed in the Search Results tab, complete the criteria as described for the Search Criteria tab when using the Database/Using Criteria correction method.
