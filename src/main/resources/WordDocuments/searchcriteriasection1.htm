Search Criteria Section

To add search criteria, follow these steps:

 

1   Specify the field you want the KFS to search on.

 

2   Choose an operator (such as 'Equals' or 'Greater Than') and a value indicating how the KFS is to identify rows in relation to the field you have selected.

3   Click add to enter your search criteria into a group.

For example, to find and make corrections to all rows in the selected file that contain account number 1031400 and a transaction date of September 1, 2006, make the following entries in the search criteria fields.

Field

Operator

Value

Account Number

Equals

1031400

Transaction Date

Equals

09/01/2006

 

Figure 601 - Edit Criteria tab

This search retrieves all transactions in the file that involve account 1031400 and have a transaction date equal to 09/01/2006. From here, you could change the account number on the transactions to 1031420.
