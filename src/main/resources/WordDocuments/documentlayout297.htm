Document Layout

The Class Code Maintenance document includes the Edit Class Code tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Edit Class Code tab definition

Title

Description

Class Code

Required when initiating a document; otherwise, display only. The unique code that identifies this class of security.

Class Code Description

Required. Your institution’s description for this class code.

Reporting Group

Required. The group to which this class of securities belongs for reporting purposes. Reporting group codes are defined by your institution. Enter the appropriate code or use the lookup  to find it.

Accrual Method

Required. When accrual processing is appropriate for individual holdings of this class of security, the method of calculating accrued income to be used. Use the lookup  to find the appropriate code.

Security Transaction Code

Required. The asset code to be linked to the security object code on the General Ledger. This code is used on transactional e-docs that increase or decrease the holdings of this class of security. Enter the appropriate code or use the lookup  to find it.

Security Transaction Code Description

Display only. Your institution’s description of the security transaction code.

Security Income Transaction Post Code

Required. The code to be used to post income activity to the KEMID for this security. Enter the appropriate code or use the lookup  to find it.

Security Income Post Code Description

Display only. Your institution’s description of the security income transaction post code.

Tax Lot Indicator

Indicates whether each KEMID acquisition of security units (holdings) are to be maintained in separate accounting (tax) lots or consolidated into one holding. This indicator affects the way gains and losses on the sale of the security are calculated.

Class Code Type

Required. Identifies a group of class codes. The options are Stock, Bond, Liability, Alternative Investment, Cash Equivalent, Pooled Investment, and Other.

Valuation Method

Required. The method by which this type of security is valued—either by an established unit value or by an established market value at a point in time (typically used for alternative investments). Use the lookup  to find the appropriate valuation method.

Active Indicator

 

Indicates whether this class code is active or inactive. Remove the check mark to deactivate a class code.
