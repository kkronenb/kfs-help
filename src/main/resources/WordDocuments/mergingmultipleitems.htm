Merging Multiple Items

To merge multiple line items:

1.     On the Active Line Items tab, check the Select check box for each item to be merged.

The system chooses the first line you checked to become the 'target' into which the other lines will be merged.

2.     In the How Many Assets field near the bottom of the screen, specify the number of assets to be created.

3.     Enter the system description.

4.     Click the merge button at the bottom of the screen.

The system combines the lines and updates the quantity on the target line. The description on the target line becomes the description of the new asset.
