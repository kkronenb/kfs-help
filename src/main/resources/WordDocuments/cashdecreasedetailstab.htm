Cash Decrease Details Tab

Cash Decrease Details tab

This tab indicates that this is a cash transaction and specifies whether it was created by the system or by a user.

Cash Decrease Details tab definition

Title

Description

Transaction Sub-Type

Display only. Indicates whether this transaction is for cash or non-cash. When you initiate this e-doc, the system sets the value to ‘Cash.’

Transaction Source

Display only. Indicates whether this transaction was generated automatically or entered by a user. When you initiate this e-doc, the system sets the value to ‘Manual Entry.’
