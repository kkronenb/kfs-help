Global Documents

Some COA documents introduced in the KFS User Guide (Account, Account Delegate, Object Code, and Sub-Object Code) and maintenance documents (Organization Reversion) allow you to assign identical values to multiple 'objects' on a single document. They are called global documents.

To initiate a global document, select it from the appropriate menu tab and enter data.

Figure 13 - Account Global document

For more information on global documents, see Account Global, Account Delegate Global, Object Code Global, and Sub-Object Code Global in the Guide to the KFS Main Menu.
