Process

When a financial document that modifies an existing asset is approved, the system adds a payment to the asset database for the given asset number. Next, a batch process loads an approved financial document into Capital Asset Builder Other G/L Transactions. Finally, the Capital Asset Office reviews the transaction and, if appropriate, applies the transaction to the asset.
