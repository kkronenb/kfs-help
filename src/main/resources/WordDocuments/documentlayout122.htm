Figure 68 - Disbursement Voucher Travel Per Diem document

The Disbursement Voucher Travel Per Diem document includes the Edit Travel Per Diem Category tab. The system automatically enters data into both the Old and New sections. Selected data fields are available for editing.

Table 52   Edit Travel Per Diem Category tab definition

Title

Description

Fiscal Year

The fiscal year when the per diem category is valid.

Per Diem Category/ Country Name

The descriptive name of the per diem category. This name appears in the Category list of the Per Diem section within the Non-Employee Travel Expense tab of the DV document.

 

Per Diem Category/ Country URL

The Web site to which users are directed when they choose the category and click the Per Diem links in the Per Diem section within the Non-Employee Travel Expense tab of the DV document.
