Endowment Transaction Code 

 >  >  >

The Endowment Transaction Code Maintenance document allows you to create and manage transaction codes (usually known as Etran codes). The system uses these codes to identify the purpose of each transaction affecting a KEMID. Like object codes, Etran codes identify categories of assets, liabilities, income, and expense. Each Etran code defined by your institution is mapped to one or more chart/object code combinations in the General Ledger. This mapping allows the system to generate appropriate General Ledger entries for your endowment transactions.

     General Ledger entries are generated by a nightly batch process.
