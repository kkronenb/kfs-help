Holding Tax Lots Tab

This tab lists the tax lots associated with the KEMID/security combination to be rebalanced. Each lot held by the KEMID is shown in a separate tab displaying the lot number in the tab header. The Holding Units and Carry Value are editable fields, so you may change the entries there.

Holding Tax Lots tab definition

Title

Description

Holding Units

The number of units of the security held by the KEMID. Only positive values may be entered. You may bring the value of units to zero. The total number of units before the change must equal the total number of units after the change.

If the units are zero, the carry value must also be zero. If the carry value is zero, the units must also be zero.

Carry Value

The carry value of the security held by the KEMID. Only positive values may be entered. You may bring the carry value to zero. The total carry value before the change must equal the total carry value after the change.

If the carry value is zero, the units must also be zero. If the units are zero, the carry value must also be zero.

Lot Number

Display only. A sequence number identifying individual acquisitions of the same security by a KEMID.
