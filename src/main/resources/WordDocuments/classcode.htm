Class Code 

 >  >  >

The Class Code Maintenance document is used to specify different categories of securities. The use of class codes allows your institution to establish security categories with as much or little detail as needed. For example, class codes 400-499 might represent stocks and 410 might represent bank stocks.

Each Class Code record associates its class code with a class code type (stocks, bonds, etc.) that determines the business rules and market value calculations the system is to apply to transactions involving securities with this class code. Each Class Code record also specifies an accrual method. The system uses this information to identify additional required fields (if any) for Security records with this class code.
