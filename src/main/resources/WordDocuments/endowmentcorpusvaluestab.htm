Endowment Corpus Values Tab

 

     This tab is displayed only in the KEMID inquiry view. It is not displayed in create or edit mode because the values are system generate, they are for information only, and they are not editable.

This tab displays the corpus value and market value data for a true endowment.

Endowment Corpus Values tab definition

Title

Description

Endowment Corpus

Display only. The value of an endowment. This value changes as a result of endowment transactions.

Current Principal Market Value

Display only. The current market value of the endowment’s principal.

Prior Fiscal Year End Corpus Value

Display only. The market value of the endowment’s principal for the previous fiscal year.
