Document Layout

 Asset Transaction Type document

The Asset Transaction Type document includes the Edit Capital Asset Transaction Type tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Edit Capital Asset Transaction Type tab definition

Title

Description

Capital Asset Transaction Type Code

A unique code to identify a capital asset transaction type.

Capital Asset Transaction Type Description

Required. A description for the capital asset transaction type.

Capital Asset Nonquantity Driven Allow Indicator

Optional. Indicates if the object sub-type code does not require a quantity.

Capital Asset Quality Subtype Required Text

Required. A description of the object sub-type codes for quantity driven capital assets.

Capital Asset Nonquantity Subtype Required Text

Required when the Capital Asset Nonquantity Driven Allow Indicator is checked. A description of the object sub-type codes for non-quantity drive capital assets.

Active Indicator

 

Indicates whether this capital asset transaction type code is active or inactive. Remove the check mark to deactivate this code.
