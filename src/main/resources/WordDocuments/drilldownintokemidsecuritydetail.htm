Drilldown into KEMID Security Detail

From the results table for the KEMID Current Reporting Group Lookup screen, you may drill down into detail for any security/registration code combination. To do so, click an underlined Security Number link. The system displays the Security Lookup screen with data for the security you chose.

   For information on the Security Lookup screen, see Security.
