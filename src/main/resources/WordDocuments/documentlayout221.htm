Document Layout

The Donor Maintenance document includes the Edit Donor Record tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Edit Donor Record tab definition

Title

Description

Donor ID

Required when initiating a document; otherwise, display only. The code that uniquely identifies a specific donor within the KFS Endowment module. The person represented by the donor ID receives the annual donor endowment statement.

First Name

The first name of the donor or the donor organization name.

Middle Name

The middle name (or initial) of the donor.

Last Name

The last name of the donor.

Address Line 1

The first line of the donor’s address.

Address Line 2

The second line of the donor’s address.

Address Line 3

The third line of the donor’s address.

City

The city in which the donor resides.

State

 

The state in which the donor resides.

Country

 

The country in which the donor resides.

Postal Code

 

The zip code or postal code for the area in which the donor resides.

Phone Number

The donor’s phone number.

Individual Salutation

The preferred salutation to be used for this donor when correspondence is directed to the donor as an individual.

Individual Mail Label

The preferred mailing label to use for this donor when correspondence is directed to the donor as an individual.

Joint Salutation

The preferred salutation to use for this donor when correspondence is directed to the donor jointly with another donor.

Joint Mail Label

The preferred mailing label to use for this donor when correspondence is directed to the donor jointly with another donor.

Development Officer

Your institution’s primary development officer assigned to this donor.

Deceased Date

If the donor is deceased, the date of the donor’s death. Enter a date or select it from the calendar .

First Corporate Contact

If the donor is a corporation, the primary contact within the corporation for information regarding contributions.

First Contact Title

If the donor is a corporation, the title of the individual listed as the first corporate contact.

Second Corporate Contact

If the donor is a corporation, the secondary contact within the corporation for information regarding contributions.

Second Contact Title

If the donor is a corporation, the title of the individual listed as the second corporate contact.

Comments

Additional comments providing further direction for anyone referencing this Donor record.

Active Indicator

 

Indicates whether this Donor record is active or inactive. Remove the check mark to deactivate this Donor record.
