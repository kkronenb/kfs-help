Financial Processing

 >

The KFS uses many supporting reference tables for central processing. Similar to the Chart of Accounts documents, the financial processing documents represent tables that define attributes and valid values within the KFS. These values tend to be for functions related to central processes, or they provide defaults for specific document types such as the Service Billing document.
