Document Layout

 Figure 236 - Account Reports document

The Account Reports document includes the Edit BC Account Reports tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Table 202   Edit BC Account Reports tab definition

Title

 

Description

Chart Code

 

The Chart of Accounts for an account to be reported.

Account Number

The account number for a pool of funds to be reported.

Reports To Chart

Required. The chart for the reports-to organization for the account to be reported. Existing charts may be retrieved from the lookup.

Reports To Organization

Required. The reports-to organization for the account to be reported. Existing organizations may be retrieved from the lookup.
