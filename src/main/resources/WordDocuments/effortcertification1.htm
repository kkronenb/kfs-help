Effort Certification

>>

The KFS Effort Certification module automates the creation of effort certification reports (also referred to as effort reports). Each of these reports certifies the percentage of effort an employee worked on a particular project represented by a KFS Contracts and Grants account.

Each employee identified as being funded by or cost-shared on a sponsored agreement has an effort certification report generated by a batch process. Each report routes for approval and may be modified to correct the distribution of effort. After being fully approved, an effort certification report generates Salary Expense Transfer documents to align the Labor Ledger and General Ledger with the newly certified effort. The fully-approved effort certification report serves as a permanent record detailing the distribution of effort for an employee for a given period.

The module also includes an Effort Certification Recreate (ECD) document that allows the user to generate effort certification reports on an exception basis after the batch process has generated reports for a specific report period.

Several special reports allow users to view effort certification reports still outstanding for approvals, view the extracted Labor Ledger data used to build effort certification reports, and identify duplicate reports.
