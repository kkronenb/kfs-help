Format Reset Fields

Tab Name

Purpose

Process Campus

Optional. Enter the campus code of the PDP processor who formatted the payments. You can also search for and select a campus using the Campus lookup .

Begin Process Date From

Optional. You may specify a date range in which the format process ran. Enter the earliest begin date you wish to search for in the 'from' field.

Begin Process Date To

Optional. You may specify a date range in which the format process ran. Enter the latest begin date you wish to search for in the 'to' field

Process ID

Optional. Enter the process ID number you want to view. Alternately you can search for the ID using the Payment Process lookup .
