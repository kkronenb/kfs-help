Accessing GL Functions

Most users access the General Ledger only to make balance inquiries. All GL inquiry functions are accessible via the Balance Inquiries submenu on the Main Menu tab.

   For more information about GL balance inquiries, see GL Balance Inquiries.

Authorized system administrators access the KFS to upload batched data to the GL for processing. All GL batch upload functions are accessible via the Batch submenu on the Administration menu tab.

   For more information about the GL batch upload functions, see GL Batch Functions.
