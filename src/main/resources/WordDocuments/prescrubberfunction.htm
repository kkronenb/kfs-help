Pre-Scrubber Function

The 'Scrubber' also includes a pre-scrubber function that can be used to complete Chart of Account code values for entries that do not contain them. This function will be used only if your institution has decided not to allow the same account number to be used on multiple Charts of Accounts (the KFS-SYS ACCOUNTS_CAN_CROSS_CHARTS_IND parameter must be set to 'N'). In this case a special step will complete the Chart of Accounts code on incoming transactions if it is blank. Note that the pre-scrubber will not correct invalid chart/account combinations (these will become 'Scrubber' errors) and will not complete the Chart of Accounts field if the supplied account number is invalid. Also, the setting of this parameter will have no impact on the function of Account maintenance documents.
