Parameter

 >  >  >  >  >

The Parameter document is used to define parameters and business rules in the KFS. A specific value of a parameter can vary based on what the parameter is used to define. Some parameters create business rules. These rules create restrictions and enforce valid values and combinations on various document types or batch processes. Other parameters simply define institution-specific values not defined elsewhere in KFS. The value may, for example, be text that the KFS is to display in a given location or it may be a simple yes or no value to turn an option on or off.

Only members of the KR-SYS Technical Administrator or KFS-SYS Manager role can create or edit Parameter documents.
