Consolidation of Records

Records are consolidated by summing the Amount fields of records with the same balance key (Employee ID, Chart, Account, Sub Account, Object Code and Position Number).

Employee totals are calculated by summing all the record amounts with the same employee ID.

If the employee total is 0, all records for that employee are dropped from the process.
