Agreement Status Code 

 >  >  >  >

The Agreement Status Code Maintenance document is used to manage KEM agreement status codes. The agreement status codes allow your institution to list unique status definitions for agreements governing a KEMID. These institutionally defined codes may be used in addition to the basic codes provided with the standard KEM model. For example, your institution’s agreement status codes might include Completed and Signed, Draft, Not Applicable, and Pending Completion.

     An agreement must have a status of completed for the account to be active.

     The assignment of an agreement status code to a KEMID agreement record results in the assignment of a default transaction restriction for the affected KEMID.
