Document Layout

The Subclass Code Maintenance document includes the Edit Subclass Code tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Edit Subclass Code tab definition

Title

Description

Subclass Code

Required when initiating a document; otherwise, display only. Identifies a subcategory within a class code for a security. This code is used for reporting purposes only.

Subclass Code Description

Required. The full description of the code.

Active Indicator

 

Indicates whether this subclass code is active or inactive. Remove the check mark to deactivate the code.
