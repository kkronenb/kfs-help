BC_Annual_Request_Imports.xls

The BC_Annual_Request_Imports.xls template applies to the Budget Construction (BC) document type and is used to import annual budget data.

The basic format of the template is shown below.

BC_Annual_Request_Imports template format

Column

Field Name

A

Chart Code*

    If your institution has set the ACCOUNTS_CAN_CROSS_CHARTS_IND parameter to NO, delete this column before using the template.
