Document Layout

The Agreement Status Code Maintenance document includes the Agreement Status Code tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Edit Agreement Status Code tab definition

Title

Description

Agreement Status Code   

Required when initiating a document; otherwise, display only. Indicates the status of the most recent account agreement documentation on file. An agreement must have a status of completed for the account to be active.

Agreement Status Code Description

Required. An institutionally defined description for this status.

Default Transaction Restriction       

Required. Identifies the types of transactions that may and may not be posted to an account—for example, income only, gifts only, no transactions, and no disbursement transactions.

Active Indicator

 

Indicates whether this agreement status code is active (Yes) or inactive (No).
