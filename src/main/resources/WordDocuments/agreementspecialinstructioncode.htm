Agreement Special Instruction Code 

 >  >  >  >

The Agreement Special Instruction Code Maintenance document is used to manage donor instructions that are unique to a gift agreement. Each code defines a unique special activity that is to be undertaken by an organization to ensure compliance with the intent of the donor. For example, a special instruction code might be to indicate that the donor wishes a percentage of the earnings go to another KEMID spending account or to return earnings to principal until a certain value is reached.

Special instructions for the KEMID may be temporary, so these codes are editable. Special instructions are used only for reporting and have no business rules attached to them.
