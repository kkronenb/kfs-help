Show Position Pick List

To view the appointment data from the position perspective, use the Show Position Pick List option in the Select Operation tab. To so so:

1   Select the Organization Sub-Tree and click the view button next to Show Position Pick List.

Figure 48 – Selecting the sub-tree for which to show appointment data

 

2   Enter your select criteria and click search

Figure 49 – Budget Construction Position Select Lookup screen

 

The  button expands the search option to the entire BC position or BC incumbent table, while the initial search screens return the subset associated with selected organization(s).  The Posn Salset w/ sync menu becomes available when you click extended.

Figure 50 – Posn Salset w/sync menu

The extended button is typically used to add new incumbents or positions, not positions that are not currently budgeted in the selected organization(s) sub-tree.

3   Click Posn Salset to select a position

Figure 51 – Click Posn Salset in the Actions column.

 

The system displays the Salary Setting by Position screen.

Figure 52 – Salary Setting by Position screen

For information about using this screen, see Salary Setting by Position.
