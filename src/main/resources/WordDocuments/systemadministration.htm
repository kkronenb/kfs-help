System Administration

This section of help provides information about using KFS system administration functions available via the user interface. All of these functions are accessible from the Administration menu tab in the Kuali Financial System (KFS) version 4.0.

   In order to work efficiently, you need to understand the basics of the KFS user interface. For information and instructions about logging on and off, navigating, understanding the components of screens, and performing basic operations in the screens, see Overview
