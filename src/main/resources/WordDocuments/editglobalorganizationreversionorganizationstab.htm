Edit Global Organization Reversion Organizations Tab

There are two ways to populate the list of organization codes that you want to change globally. One is by entering or selecting one organization code at a time from the normal Organization Code lookup . The other is by using the Look Up / Add Multiple Organization Code Lines  to return multiple values.

   For information about the difference between the two types of lookup, see Field Lookup and Multiple Value Lookup.
