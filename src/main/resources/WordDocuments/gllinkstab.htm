GL Links Tab

The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing. Note that you may specify multiple GL links for each Etran code.

GL Links tab definition

Title

Description

Chart Code

 

Required when adding a new GL link; otherwise, display only. The KFS chart code to be used for generating GL entries for transactions containing this Etran code. Select from the list or use the lookup  to find the appropriate chart code.

    In any given transaction, the Etran code and KEMID specified must be associated with the same chart.

Object Code

 

Required. The General Ledger object code to be used for generating GL entries for transactions with this Etran code. Enter a valid object code or use the lookup  to find it.

    The object specified here must be valid for the chart specified in the Endowment Transaction Code tab. The object must also be of the type specified in the Transaction Code Type field in the Endowment Transaction Code tab. For example, if the entry in the Transaction Code Type field is Income, the object code specified here must also be associated with the Income object code type.

Object Name

Display only. The descriptive name of the object code.

Active Indicator

 

Required. Indicates whether this GL link is active or inactive. Remove the check mark to deactivate this GL link for this Etran code record.

Click  to add a GL link record to this Etran code record.
