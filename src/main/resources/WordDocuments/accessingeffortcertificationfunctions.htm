Effort Certification

Accessing Effort Certification Functions

The Effort Certification functions are organized on the KFS menus as follows.

•      One e-doc—Effort Certification Recreate—is listed in the Administrative Transactions submenu on the Main Menu tab.

•      To facilitate research, the Main Menu provides an effort-certification-specific search in the Custom Document Searches submenu.

•      The Reports submenu on the Main Menu provides access to a number of standard effort certification reports.

•      The Maintenance menu tab offers access to four Effort Certification maintenance functions that allow you to extract information from the Labor Ledger to generate effort reports using KFS batch jobs. These e-docs are accessible via the Effort Certification submenu.
