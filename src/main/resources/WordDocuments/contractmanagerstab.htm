Contract Managers Tab

Note that multiple Campus Code/Contract Manager combinations may be assigned to the commodity code by clicking add after entering data for each combination.

Contract Managers tab definition

Title

Description

Required. The campus for the commodity code. Existing campus codes may be retrieved from the list or from the lookup .
