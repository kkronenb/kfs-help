Account Delegate Model

 > >>> >

The Account Delegate Model document allows you to create a model for the Account Global documents so that you can define a common delegation structure for a group of accounts easily. After the models are created, you may use them from the Create New Global from Model menu option.

   For information about the Create New Global from Model option, see Creating New Global from Model.
