Deposit Header Tab

This tab contains banking information for this deposit.

Deposit Header tab definition

Title

 

Description

Bank

 

Required. Enter the bank code to identify the bank in which the deposit is to be made or select it from the Bank Code lookup .

Bank

Account

 

Required. Enter the bank account number to make a deposit to or select it from the Bank Account lookup .

Deposit Type

Display-only. It indicates whether the type of deposit chosen was interim or final.

Deposit#

Optional. Enter the number assigned by the CMD user to identify this deposit.
