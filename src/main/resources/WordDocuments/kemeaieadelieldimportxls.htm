KEM_EAI_EAD_ELI_ELD_Import.xls

The KEM_EAI_EAD_ELI_ELD_Import.xls template applies to the Asset Increase (EAI), Asset Decrease (EAD), Liability Increase (ELI), and Liability Decrease (ELD) e-docs. For each of these e-docs, this template is used to import transaction lines.

     Be sure to select the appropriate section of this template. The KEM_EAI_EAD_ELI_ELD_Import.xls template contains two sections, one for cash-based transactions and the other for non-cash-based transactions. To select the appropriate section of the template, click a tab at the bottom of the Excel spreadsheet.
