KEMID Submenu Group

The KEMID submenu group contains e-docs that allow you to maintain basic attributes used in your institution's KEMID records.

Endowments submenu, KEMID group

Document

 

Description

Agreement Special Instruction Code

Stores donor instructions that are unique to a gift agreement.

Agreement Status Code

Defines your institution's unique status agreement codes for agreements governing activities of a KEMID.

Agreement Type Code

Defines the types of agreements that exist for the establishment and support of KEMIDs.

Close Code

Defines the reasons a KEMID may be closed.

Combine Group Code

Defines reporting group codes.

Donor Record

Identifies a donor record for the purpose of creating periodic or annual donor statements.

Donor Statement Code

Indicates the type of annual statement to provide for a given donor.

Fee Method

Allows you to specify sets of fee calculations to be used by the system to assess fees against KEMIDs

Purpose Code

Identifies an allowable use of funds; may indicate a very specific limitation or a broad range of uses depending on the donor wishes for a gift fund.

Responsible Administration Code

Identifies the section or department in the endowment management and accounting area of the institution that is responsible for oversight of a KEMID and its activity.

Source of Funds Code

Identifies a standard source of funding for KEMIDs.

Tickler Type Code

Defines a unique reason a person or department may wish to be notified of an upcoming event or action to be taken.

Transaction Restriction Code

Identifies a specific rule regarding allowable transactions for KEMIDs.

Type Code

Identifies a group of KEMIDs for reporting purposes.

Type Restriction Code

Indicates how funds are to be used according to the agreement establishing a KEMID.

Use Criteria Code

Identifies a specific restriction that a donor has placed on the use of the funds and or earnings of a gift.
