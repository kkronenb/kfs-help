Edit Criteria Tab

The Edit Criteria tab is where you identify the rows of data you want to make changes to and specify the desired changes. To make corrections you can create one or more groups of search criteria and corresponding modification criteria. Each group can have multiple search and modification criteria.

The Edit Criteria tab contains the Search Criteria section and the Modification section.
