Effort Certification > Maintenance Functions > Report Type

Document Layout

The Report Type document includes the Edit Report Type tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Report Type tab definition

Title

 

Description

Report Type Code

A unique code identifying the pay frequency for a group of employees.

Examples include:

10 = 10 pay reporting
12 = 12 pay reporting
BI = Biweekly reporting
SU = Summer Reporting

Report Description

Required. A text description for this report type.

Active Indicator

 

If checked, this report type is valid and may be used on the Report Definition or Report Earn Paygroup documents. If unchecked, this report type is inactive and may not be used.
