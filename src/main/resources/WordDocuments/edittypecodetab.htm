Edit Type Code Tab 

The Type Code Maintenance document includes the Edit Type Code tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Edit Type Code tab definition

Title

Description

Type Code

Required when initiating a document; otherwise, display only. A code used by your institution to group KEMIDs for reporting purposes.

Type Code Description

Required. Your institution’s description of this type code.

Default Cash Sweep Model

The default cash sweep model associated with this type code. This value is automatically displayed in the Cash Sweep Model field for each new KEMID that uses this type code.

Default Cash Sweep Model Description

Your institution’s description of the default cash sweep model.

Default Income ACI Model

The default income ACI (automated cash investment) model associated with this type code. This value is automatically displayed in the Income ACI Model field of each new KEMID that uses this type code.

Default Income ACI Model Description

Your institution’s description of the default income ACI model.

Default Principal ACI Model

The default principal ACI model associated with this type code. This value is automatically displayed in the Principal ACI Model field of each new KEMID that uses this type code.

Default Principal ACI Model Description

Your institution’s description of the default principal ACI model.

Active Indicator

 

Indicates whether this type code is active or inactive. Remove the check mark to deactivate this type code record.
