Holding Tax Lot Maintenance Document

From the results table for the KEMID Current Tax Lots Lookup screen, you may access the Holding Tax Lot Maintenance Document. In this document you may view details of a tax lot and, if desired, change the current and prior accrual values.

To access the maintenance e-doc for a tax lot, click the edit link in the Actions column for the lot. The system displays the Holding Tax Lot Maintenance Document.

Holding Tax Lot Maintenance Document

The Holding Tax Lot Maintenance Document contains one unique tab—Holding Tax Lot Balance—in addition to the standard tabs.

    For more information about the standard tabs, see Standard Tabs in the Overview of the Kuali Financial System.

This tab contains detail about the tax lot. All fields are display only, except for Current Accrual and Foreign Tax Withheld, both of which you may change in order to change the calculations for the tax lot.
