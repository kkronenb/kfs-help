Core Modules

The core modules include the following components of the Kuali Financial System:

•      General Ledger

•      Chart of Accounts module

•      Vendor module

•      Pre-Disbursement Processor module

•      System functions and e-docs that pertain to multiple modules

   Although the Financial Processing module is one of the “core” KFS modules, institutions can and occasionally do choose not to implement it initially. Consequently, information about that module is presented in Purchasing/Accounts Payable.

   In order to work efficiently in the KFS screens, you need to understand the basics of the user interface. For information and instructions on logging on and off, navigating, understanding the components of screens, and performing basic operations in the screens, see Overview.
