Adding a Group

You can make multiple changes to a file based on different search criteria by adding additional groups. To add another group, click the add button to the left of Add Groups.

The system creates another entry in this tab where you can enter more search and modification criteria.

For example, to add another criterion that selects the transactions with a project code of ABC, you may specify it in the second group by selecting the Project Code field and setting the value to 'ABC'. To correct the value by changing it from 'ABC' to 'XXX,' select 'Project Code' from the Field list in the Modification Criteria section and enter the value 'XXX'.
