Asset Information Subtab

Asset Information tab

The Asset Information subtab displays additional attributes of the particular asset. The information on this subtab is view only.

Asset Information subtab definition

Title

Description

Organization Owner Account Number

The account for the owner that is responsible for the asset.

Asset Type Code

The code used to classify types of assets grouped by categories.

In-Service Date

The date the asset is placed in service and becomes eligible for depreciation.

Asset Description

A detailed description of the asset.

Organization Owner Chart Of Accounts Code

The chart code for the owner account that is responsible for the asset.

Vendor Name

The company or person who has sold the asset to the institution and to whom the first payment was made.

Manufacturer

The name of the person or the company that manufactured the asset.
