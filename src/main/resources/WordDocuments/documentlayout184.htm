Document Layout

Type Restriction Code Maintenance document

The Type Restriction Code Maintenance document includes the Edit Type Restriction Code tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Edit Type Restriction Code tab definition

Title

Description

Type Restriction Code

Required when initiating a document; otherwise, display only. A code associated with the Income and/or Principal side(s) of a KEMID. This code affects transaction processing for true endowments.

Type Restriction Code Description

Required. Your institution’s description of this type restriction code.

Permanent

Indicates whether this type restriction code applies to funds that are part of a true endowment. Check the box to indicate that this code applies to funds that are part of a true endowment.

Active Indicator

 

Indicates whether this type restriction code is active or inactive. Remove the check mark to deactivate a Type Restriction code record.
