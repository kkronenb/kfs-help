Edit Calculated Salary Foundation Tracker Override tab definition

Title

 

Description

University Fiscal Year

Required. The university fiscal year. Existing fiscal years may be retrieved from the lookup .

Chart Code

 

Required. The unique identifying code assigned to a chart for the override. Existing chart codes may be retrieved from the lookup .

Account Number

Required. The account number for the override. Existing account numbers may be retrieved from the lookup .

Sub-Account Number

Required. The sub-account number for the override. Existing sub account numbers may be retrieved from the lookup .

Object Code

 

Required. The object code for the override. Existing object codes may be retrieved from the lookup .

Sub-Object Code

 

Required. The sub-object code for the override. Existing sub object codes may be retrieved from the lookup.  

Position Number

Required. The position number for the override.

Employee ID

Required. The employee ID for the override. Existing employee ID's may be retrieved from the lookup .

CSF Amount

Required. The position CSF amount for the override.

CSF FTE

Required. The CSF full time equivalent value for the override.

CSF Time Percent Time

Required. The CSF percent time value for the override.

CSF Funding Status Code

Required. The CSF funding status code for the override.

CSF Delete Code

Required. The CSF delete code for the override.

Active Indicator

 

Required. Indicates whether this override is active or inactive. Remove the check mark to deactivate this override.
