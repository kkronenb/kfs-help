Updating Using the Adjust  % or Flat Tool

The Adjust: % /Flat tool allows you to apply a flat amount or a percentage increase or decrease to one or all the funding lines. The tool is displayed only when the line is editable and the funding line has a base budget in CSF. To use the tool, follow these steps:

 

1   Select the type of change desired from the drop-down menu and enter the amount.

 

Figure 150 – Selecting the type of change desired

2   Click apply.

 

Figure 151 – Information entered

 

The system automatically calculates the new amount and the hourly rate.

 

Figure 152 – Results of system calculations

3   Click save.
