Group_Import.xlxs

The Group_Import.xlxs template applies to the Travel Authorization, Travel Authorization Amendment and Travel Reimbursement documents. For each of these documents, this template is used to import group travelers.

The basic format of the template is shown below.

Group_Import template format

Column

Field Name

A

Traveler Type Code* (Employee, Student, Customer or Other)

B

Group Traveler ID (KIM Principal Name or AR Customer ID)

C

Name*
