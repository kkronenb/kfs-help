Adding One Account at a Time

1.     Complete the Edit Global Delegate tab.

2.     To add one account at a time, enter the values or use the Account lookup .

3.     Search for an account and click the  link associated with the account you want to select.

     For information about field lookup and multiple value lookup, see Field Lookup and Multiple Value Lookup.

4.     Click  to add the account to the list.
