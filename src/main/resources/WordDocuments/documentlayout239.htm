Document Layout

The Labor Position Object Group Code Maintenance document includes the Edit Position Object Group tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Edit Position Object Group tab definition

Title

Description

Labor Position Object Group Code

A unique code for a labor position object group.

Labor Position Object Group Name

Required. The familiar name for the labor position object group.

Active Indicator

 

Indicates whether this labor position object group code is active or inactive. Remove the check mark to deactivate this code.
