Labor Object Code Benefits

 >>>>

The Labor Object Code Benefits Maintenance document associates salary Object Codes with a Labor Benefit Type Code. One salary code may have several Labor Benefit Type Codes associated with it, each represented by a different row in this table. This information is used to determine which benefits should be associated with a given salary object code for the purpose of Salary Transfer documents.

For more information about Labor Benefit Type, see Labor Benefits Type.
