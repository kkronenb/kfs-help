Effort Certification Recreate

>>>>

The Effort Certification Recreate (ECD) document allows an authorized user to manually extract data from the Labor Ledger for a single employee and recreate an effort report for a specified fiscal year and reporting period.

The end result of this process is an effort certification report that functions as any other. Use of this document is restricted to users with the Contracts and Grants Manager role.

For information about the Effort Certification Document (A21), see Effort Certification Document (A21).
