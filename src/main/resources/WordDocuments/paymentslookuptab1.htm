Table 70   Payments Lookup tab definition

Title

Description

To view the payment lookup for this Asset

Link (Click here) to the Asset Payments lookup for this asset.

Clicking this link causes the system to display the Asset Payment Lookup screen with the Asset Number field pre-populated with the asset number of the asset the user is working with. The results table below the form contains all payments that have been processed.

If more than ten payments exist for the asset, the user may view all payments via the Asset Payment Lookup. For more information on the lookup form, see Asset Payment Lookup.
