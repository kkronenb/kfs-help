Document Layout

The Non-Resident Alien Tax Percent document includes the Edit Non Resident Alien Tax Percent tab. The system automatically enters data into both the Old and New sections. Selected data fields are available for editing.

Edit Non Resident Alien Tax Percent tab definition

Title

Description

Income Class Code

A code to reflect the type of disbursement activity, applied to a category to the services or goods for which funds are being disbursed relative to non-resident alien tax.

Income Tax Type Code

A unique code to indicate if the non-resident alien tax is state or federal.

Income Tax Percent

The non-resident alien tax percentage to be applied for the combination of income class code and income tax type.

Active Indicator

Indicates whether this non-resident alien income tax type code is active or inactive. Remove the check mark to deactivate this code.
