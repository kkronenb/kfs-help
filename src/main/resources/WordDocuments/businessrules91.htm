Business Rules

•      The KFS-LD FISCAL_YEAR_SELECTION parameter is used to compute fiscal year plus 1.

•      Exclude the Labor Ledger Balance record if the balance type is not 'AC.

•      The file created is called ld_balance_forwards.data.

•      Exclude the Labor Ledger Balance record if the chart of accounts code on the account in the Labor Ledger Balance table does not match the chart of accounts code in the Account table.

•      Exclude the Labor Ledger Balance record if the sub-fund group code on the account is not found in the Sub-Fund Group table.

•      Exclude the record if the fund group code is not found in the KFS-LD SUB_FUND_GROUPS parameter.

•      Accumulate various totals during the process and print a summary report at the end of the process.
