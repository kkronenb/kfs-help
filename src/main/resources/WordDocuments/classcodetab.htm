Class Code Tab

Class Code tab

The Class Code tab identifies the various security classes (if any) to be used as selection criteria for calculating fees.

     This tab is available for your use only if the Fee Type value is ‘B’ (balance) in the Fee Method tab.

     If the Select by Class Code box is checked in the Fee Method tab, the Class Code tab must contain at least one class code for which the Include box is checked.

Class Code tab definition

Title

Description

Class Code

Required if class code is to be used as a criterion when calculating fees; otherwise, not used. Specifies a class code to be used. Each class code denotes a different category of securities.

Class Code Description

Your institution’s description of the class code.

Include

Indicates whether this class code is to be used as a basis for fee calculation. Uncheck the box to discontinue using this class code for the fee calculations for this fee method.
