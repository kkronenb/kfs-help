A new proposal number is created for all requests for external funding (new proposals, competing renewals, continuations and supplements) either manually or via the system. In general, the proposal document is used in conjunction with a funding request.

The fully-approved proposal document populates boxes in the Award document when the related award is received and recorded in the financial system. For instance, the PD name captured in the proposal document populates the award document when it is set up.
