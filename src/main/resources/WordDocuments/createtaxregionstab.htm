Create Tax Regions Tab

Create Tax Regions tab definition

Title

 

Description

Tax Region Code

The tax region code associated with this tax type.

Tax Region Name

Required. Familiar title for the specific tax region associated with a tax type.

Tax Region Type Code

The code for the tax region type associated with this tax region.

Liability Chart Code

Required. The chart code for the organization that owns the liability account number to which transactions will post. Existing chart codes may be retrieved from the lookup .

Liability Account Number

Required. The account number to which liability transactions resulting from tax calculations will post. Existing account numbers may be retrieved from the lookup .

Liability Object Code

Required. The liability object number to which transactions resulting from tax calculations will post. Existing object codes may be retrieved from the lookup .

Tax Region Use Tax Indicator

Indicates whether use tax applies to this tax region. Remove the check mark if no use tax applies.

 

Active Indicator

 

Indicates whether a tax region code is active or inactive. Remove the check mark to deactivate the tax region code.
