Document Layout

When you click create new in the top right corner of the Tickler Lookup screen or click the edit link for any row in the search results display, the system displays the Tickler Maintenance Document screen. If you are creating a new tickler, the screen is displayed in create mode (as shown below). If you are editing an existing tickler, the screen is displayed in edit mode.

The Tickler document contains five unique tabs— Tickler, Tickler KEMID, Tickler Security, Tickler Recipient Person, and Tickler Recipient Group—in addition to the standard tabs.

    For more information about the standard tabs, see Standard Tabs.
