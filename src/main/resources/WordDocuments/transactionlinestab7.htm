Transaction Lines Tab

Transaction Lines tab

This tab contains one or more lines that specify the KEMID(s) affected, including all relevant financial details for the transaction as it applies to each KEMID specified.

Transaction Lines tab definition

Title

Description

KEMID

Required. The KEMID to which this transaction applies. Enter the KEMID to which this liability decrease applies or use the lookup  to find it. The system then displays the description of the KEMID you specified.

Etran Code

 

Identifies the type of processing required for this transaction.

If the Transaction Sub-Type is ‘Cash,’ you cannot enter an Etran code.

If the Transaction Sub-Type is ‘Non-cash,’ you must enter an Etran code. The Etran code for the security must have an appropriately identified General Ledger object code record—one that matches the chart for the KEMID-associated General Ledger account in the transaction line. The Etran code must have a type of expense or (more rarely) income. Enter the code or use the lookup  to find it.

Line Description

Describes this transaction. Enter text as desired.

Income or Principal

Required. Indicates whether this transaction pertains to income or principal. Enter the appropriate value from the list.

Transaction Amount

Required. When a transaction line is added, the transaction amount must equal the number of units/shares because liabilities are priced at $1 per unit.

Transaction Units

Required. Indicates the number of security units affected by the transaction. Enter a positive number.

Actions

Click the button to perform an action on this line. When you are adding a new line, only the add button is displayed. After you have added a line, the refresh, bal inquiry, and delete buttons are displayed.

Total Income Amount

Display only. Indicates the amount of income in all transaction lines.

Total Principal Amount

Display only. Indicates the total amount of principal in all transaction lines.

Total Income Units

Display only. Indicates the total number of income units in all transaction lines.

Total Principal Units

Display only. Indicates the total number of principal units in all transaction lines.

   Buttons on this tab allow you to add, import data, refresh tax lot information, delete lines and perform balance inquiries on the KEMID(s) specified. For information and instructions, see Working in the Transaction Lines Tab above.
