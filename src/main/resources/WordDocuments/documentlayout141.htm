Document Layout

The Agency Type Code document includes the Edit Agency Type Code tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Agency Type Code tab definition

Title

 

Description

Agency Type Code

A code to designate the type of agency.

Examples include
F = Federal
S = State
I = Commercial/Industrial.

Agency Type Description

Required. The description of this agency type or the category of business in which that the agency operates. Examples include 'Federal,' 'Commercial/Industrial,' 'Non-Profit,' or 'Other Government.' Up to thirty characters.

Active Indicator

 

Indicates whether this agency type code is active or inactive. Remove the check mark to deactivate this code.
