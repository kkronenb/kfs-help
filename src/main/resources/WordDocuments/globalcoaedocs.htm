Global COA E-Docs

Some COA documents (Account, Account Delegate, Account Delegate from Model, Object Code, Sub-Object Code, and Organization Reversion) allow you to assign identical values to multiple 'objects' on a single document known as a global document.

To initiate a global document, select the desired document from the Chart of Accounts menu group on the Lookup and Maintenance submenu on the Main Menu tab and enter the appropriate data.

   For more information about a specific global document, see the subsection for that e-doc.
