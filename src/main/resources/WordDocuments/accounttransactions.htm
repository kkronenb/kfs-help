Account Transactions

>>>

The Account Transactions report provides the details of the individual transactions shown in summary in the Account Status report. The transactions include actual, budget, and outstanding encumbrances (internal, external and pre-encumbrances).

The report features the following:

•      Transaction descriptions and document references are included.

•      Object code totals are shown consistent with the Account Status report.

•      There are separate sections for actual and commitment activity
