Document Layout

Appointment Funding Duration document

The Appointment Funding Duration document includes the Edit BC Duration tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Edit BC Duration tab definition

Title

 

Description

Appointment Duration Code

A unique code that identifies the appointment duration of a leave of absence for a salary.

Appointment Duration Description

Required. A description of the appointment duration code, which usually includes a description of the type and duration of the leave of absence.

Example: Sabbatical Leave: 1st Semester.

Active Indicator

 

Indicates whether this appointment duration code is active or inactive. Remove the check mark to deactivate this code.
