Performing Common Maintenance E-Doc Operations

All of the maintenance documents are viewed and maintained in a similar fashion. Selecting the desired table from the maintenance menu takes you to the lookup screen. From that screen you may create a new document by clicking the create new link or search the table to view, edit, or copy a table value. These common operations are explained below, using the Account Type document as an example.
