Notes and Attachments Tab

The Notes and Attachments tab displays user notes, attachments, or system-generated information about the document. The number of notes and/or attachments is indicated on the tab.

Size and File Type Restrictions for Attachments

The maximum size of attachments is 5 megabytes by default, but your institution may change that limit. The system imposes no restrictions on types of files that may be attached.
