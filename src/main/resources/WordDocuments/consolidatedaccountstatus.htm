Consolidated Account Status

 > >>

   For information about the standard features of KFS reports, see Standard Reports.

The Consolidated Account Status report groups all accounts for a given chart, fund, sub fund, and responsibility center. The format is somewhat similar to the Account Status Report, discussed earlier in this section. Current budget, year-to-date actual, open encumbrance, and a remaining budget balance amounts are displayed at the consolidation level for each account.
