Document Layout

The Organization Options document includes the Edit Organization Options tab, the Edit Organization Invoice Information tab, the Edit Organization Remit To Address tab, and the Edit Organization Phone tab. The system automatically enters data into both the Old and New sections. Selected data fields are available for editing.
