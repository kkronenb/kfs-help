Maintenance Menu > Budget Construction > Organization Reports
Example

The University of Kuali needs to reorganize Organization A. It currently reports to Organization B, but it needs to report to Organization C starting with the next fiscal year. All accounts reporting to Organization A need to keep reporting to Organization A, but all accounting and reporting activities will now come under Organization C's control. The Budget Construction Genesis process has been previously run to initialize the next year budget, so a maintenance user uses a Budget Construction Organization Reports maintenance document to update the reports-to organization mapping for Organization A, changing the Reports To Organization from Organization B to Organization C. The Responsibility Center Code may also need updated to be the same values used by Organization C. Any organizations reporting to Organization A may also need the Responsibility Center Code adjusted.
