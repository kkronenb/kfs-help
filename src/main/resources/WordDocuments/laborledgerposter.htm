Labor Ledger Poster

The Labor Ledger Poster writes valid entries to the Labor Ledger Entry table, updates balances in the Labor Ledger Balance table and summarizes the entries for posting to the Labor Ledger.

Several reports are generated by the labor poster process. Note that the following reports are combined into a single report (laborPosterJob) in the default configuration but are shown individually below for clarity.

Labor Poster Error Listing (labor_poster_main_error_list)

Labor Poster Summary Report (labor_poster_gl_summary)

Labor Poster Report (labor_poster_main)

Labor Poster Ledger Report (labor_poster_main_ledger)

Labor Poster Output Summary (labor_poster_output_summary)

Labor Actual Balance Summary (labor_summary_act)

Labor Budget Balance Summary (labor_summary_bud)

Labor Encumbrance Summary (labor_summary_enc)
