Barcode Inventory Error Document

The error report indicated by a document ID can be accessed in either of two ways:

1  Either click the action list button in the top left corner of the document display,

or click the doc search button in the top left corner of the document display, and then enter the document ID in the Document/Notification ID field and click search.

 

Figure 521 - Barcode Inventory Error Documents in the action list

Figure 522 - Barcode Inventory Error Document Lookup

2  Click on the document ID link in the action list or search results table.

The system displays the Barcode Inventory Error document.

The Barcode Inventory Error document consists of the standard tabs plus two additional tabs—Barcode Inventory Error(s) and Global Replace.
