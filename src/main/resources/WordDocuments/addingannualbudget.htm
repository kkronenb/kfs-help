Adding Annual Budget

To add an annual budget:

1   Click request import from the Budget Construction Selection screen.

Figure 59 – Location of request import button

 

 

2   Browse your desktop, select your file, set the file type, separator, and delimiter values, and click submit.

Figure 60 – Specifying a file and delimiters

 

The system displays an open file window for the PDF log file.

Figure 61 – Window for opening the log file

 

3   View the log file.  The log shows the records which are rejected. Review the reject reasons.

Figure 62 – Sample log file

 

4   Load the BC document.  Note that the requested budget columns have been updated for the accepted records.

Figure 63 – Updated budget columns in BC document
