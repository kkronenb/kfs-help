CSF Funding Status Code

Institution-defined. Indicates whether the CSF line has budget commitments.

  If used, the Budget Construction module must include the following statuses (required by the 'genesis' budget initiation process):

-: Active CSF line.
L: On leave

Institution-defined. Indicates whether the CSF line is active or inactive.

  If used, the Budget Construction module must use '-' to indicate active status.
