Document Header

As in other KFS documents, the header section contains basic identification and status information about the document. Note that the initiator is KFS—a username that identifies the KFS system itself. This value reflects the fact that Effort Certification documents are generated automatically via a batch process rather than initiated manually by a user.

   For information about the standard e-doc header, see Document Header.
