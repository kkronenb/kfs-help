KEMID

 > > > >

The KEMID e-doc allows you to create and work with the KEMID record that identifies a specific set of funds held by your institution as an endowment or a specific set of funds functioning as an endowment. KEMID records contain information that specifies the sort of activity that may take place for the KEMID and that specifies how and when automated activity is to occur.

When you select KEMID from the menu, the system displays the KEMID Lookup screen.

KEMID Lookup screen

On this screen, you have two options:

•      To set up a new KEMID, click the create new button in the upper right corner of the screen. The system displays the screen in create mode. Complete the fields as indicated in Document Layout below.

•      To display information for one or more existing KEMIDs, enter search criteria in the fields on this screen as explained in the table below and click search.

KEMID Lookup definition

Title

 

Description

KEMID

The KEMID for a record you want to work with. A KEMID identifies a specific set of funds held by your institution as an endowment or a specific set of funds functioning as an endowment.

Short Title

The short title associated with the KEMID you want to work with.

Type

The reporting group to which the KEMID belongs.

Purpose

The purpose of the KEMID you want to work with.

Responsible Department

The department responsible for administering the KEMID with which you want to work.

Transaction Restriction

The transaction restriction associated with the KEMID you want to work with.

Closed Indicator

Open/closed status of the KEMID you want to work with.

•      After you enter search criteria and click search, the system displays a list of matching records at the bottom of the screen.

Search results

KEMID Search results definition

Title

Description

KEMID

A specific set of funds held by your institution as an endowment or a specific set of funds functioning as an endowment.

Short Title

An abbreviated title for internal reporting use.

Type

A code identifying the group to which this KEMID belongs for reporting purposes. Your university’s groupings might include University General Endowed Funds, Discretionary Funds, and Expendable Funds.

Purpose

A code identifying the allowable use(s) of this endowment fund. Examples at your institution might include Scholarships, Professorships, Chairs, and/or General Use.

Responsible Department

A code identifying the department or group responsible for administering the KEMID.

Transaction Restriction

A code identifying the set of rules governing allowable transactions for the KEMID.

Closed Indicator

Indicates whether this KEMID is open or closed.

To edit or copy one of these records, click the  or link in the Actions column.

From the results table, you may drill down into detail for a KEMID. To do so, click an underlined KEMID. The system displays the KEMID Inquiry screen.

   To see images of the KEMID Inquiry screen in three parts, see Drilldown into KEMID Inquiry Screen. For information about the fields on the Inquiry screen, see Document Layout below.
