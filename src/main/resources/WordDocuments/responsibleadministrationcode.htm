Responsible Administration Code 

 >  >  >

The Responsible Administration Code Maintenance document allows you to manage codes that represent the department or group responsible for administering a KEMID.

     This code does not represent the department or organization that is the beneficiary of the funds. It represents the group that administers the endowment.
