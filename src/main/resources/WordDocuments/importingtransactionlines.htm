Importing Transaction Lines

If you have a number of transaction lines to enter, you may create a .csv file containing the information and import it into the Corpus Adjustment e-doc. To add multiple transaction lines from a file, follow these steps:

1.     Retrieve the data import template from KEM_ECA_Import.xls in the Overview of the Kuali Financial System.

2.     Create a .csv file and import it via the  button according to the instructions in Creating a .csv File in the Overview of the Kuali Financial System.
