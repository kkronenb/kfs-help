Managing the Budget Line

Adding a Budgeted Line

To add a new budget line, enter the required fields on the Revenue or Expenditure tab and click add.

Figure 103 – Enter data and click add.

Deleting a Budgeted Line

To delete an existing budgeted line, click the delete button on the line.

Figure 104 – Click delete

Inquiring about Balances

To inquire about balances, click the bal inquiry button. The standard Balance Inquiry Report screen is displayed that you can inquire about the account balances.

Figure 105 – Standard Balance Inquiry report

Salary Setting

To access the Quick Salary Setting screen, click salary setting.

The system displays the Quick Salary Setting screen.

Figure 106 – Quick Salary Setting screen

For information on using this screen, see Quick Salary Setting.

The salary setting total must equal the requested amount.  Always use the salary setting to derive the total requested amount at the object code level.
