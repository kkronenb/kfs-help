Pulling up the Document

The pull up button is available only to users assigned the BC processor role somewhere in the account’s organization hierarchy and is available only if the document’s current level is below the user’s approval level.

To pull up a document:

Click pull up.

Figure 92 – Location of the pull up button

The system grants you edit access and changes the pull up button to push down.

When the document is pulled up to the higher-level user, the lower l-level user may not access to the document until the higher-level user pushes the document down to the lower level.
