Cashiering Transaction Checks Tab

When you have the cashiering checks entered into the Cashiering Transaction tab, the checks are listed in the Cashiering Transaction Checks tab so you may deposit these checks along with the checks associated with the CR documents.

Title

 

Description

Description

Display-only. The description for the cashiering transaction check available for deposit.

Check Date

Display-only. The check date for the cashiering transaction check available for deposit.

Check Amount

Display-only. The check amount for the cashiering transaction check available for deposit.
