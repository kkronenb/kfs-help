Effort certification reports not fully approved (that is, documents not in 'Final' or 'Processed' status) are displayed on this report by chart and organization. The person, fiscal year and report number of the outstanding report are also displayed. To open a specific effort report, click the document number. This report also displays the name of the next approver.

Note that an effort certification report is considered to be associated with an organization if it includes one of that organization's accounts.
