Endowment > Standard Transaction E-Docs > Cash Decrease

Example

A permanently endowed KEMID owns 100 shares of a pooled long term fund represented by security ID 99PLTF123. On December 1 a fee is assessed of $1.00 per share held. To record this fee into the endowed KEMID, a Cash Decrease e-doc is initiated. The e-doc subtracts $100 of cash and includes an Etran code that indicates investment fees. Since this expense is tied to security 99PLTF123, the user enters the security ID in the optional Security Details tab.
