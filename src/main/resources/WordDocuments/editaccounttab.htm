Edit Account Tab

In the Edit Account tab, specify the chart and account that the sub-object code belongs to. There are two ways to populate the accounts that you want to select in the Edit Account tab. One is by manually entering or selecting one account at a time from the normal Account lookup , the other is by using a special multiple value lookup called the Look Up / Add Multiple Account Lines  to return multiple values.

Edit Accounts tab
