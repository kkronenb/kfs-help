Generate Ticklers Process

The Generate Ticklers process creates and delivers ticklers (electronic reminders) in the form of simple routable e-docs.Â  It creates these e-docs based on information in Tickler records whose content is due to be delivered on the current date. It also routes the generated e-docs as FYI to the person(s) and/or group(s) identified in the associated tables for the ticklers. Finally, the process places an action item in each recipient's action list.

All Tickler e-docs generated by this process are assigned a routing status of Final. Because these e-docs are FYI, no action is required on the part of the recipients, and each recipient is responsible for deleting tickler(s) from his or her action list.

This process must be run daily but may be run at any point in the overall batch processing sequence.
