Tax Region [Counties, States, Postal Codes] Tab

Figure 249 - Tax Region States tab

Multiple states may be applied to a tax region.

The fields in this tab vary depending on the type of region, and required fields vary as well. The table below lists  the fields for state regions only; other types of regions have siimilar fields for postal code, county, etc.

Table 216 Tax Region State tab definition

Title

 

Description

State Abbreviation

Required. Identifies the state applicable to a specific tax region and tax type. Existing state abbrevieations may be retrieved from the lookup.

Country Code

Required. Identifies the associated country. Existing country codes may be retrieved from the lookup.

Active Indicator

 

Indicates whether this code is active or inactive for the tax region. Remove the check mark to deactivate the country, state, or postal code for this tax region.
