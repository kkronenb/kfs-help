Payment Medium

 >>>>

The Payment Medium document defines the codes used to describe the type of payment to be applied to Accounts Receivable invoices such as check, credit card, wire, etc. Each payment medium code drives accounting rules on the Cash Control document.

Warning

Changes, additions, and deactivation of payment medium values should occur only with extreme caution and after extensive testing.
