Collector XML Upload

>>> >

When you select the Collector XML Upload option from the Administration menu tab, the system displays the GL Collector (XML format) Batch Upload document. This document allows you to upload XML input files input into the General Ledger. This upload page functions exactly the same as the Collector Flat File Upload.

For information about the Collector Flat File Upload, see Collector Flat File Upload.
