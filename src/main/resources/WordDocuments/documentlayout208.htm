Document Layout

The Cash Sweep Model Maintenance document includes the Edit Cash Sweep Model tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Edit Cash Sweep Model tab definition

Title

Description

Cash Sweep Model ID

Required when initiating a document; otherwise, display only. The unique identifier for this cash sweep model.

Cash Sweep Model Name

Required. Your institution's name for this cash sweep model.

Sweep Income Cash Limit

Required. The smallest amount of uninvested income cash your institution wants to hold in KEMIDs that use this model.

Income Sweep Investment

Required. The security ID of the security that is to be used for the sweep investment of income cash. Enter the appropriate security ID or use the lookup  to find it.

Income Sweep Investment Description

Display only. Your institution's description of the income sweep investment.

 

Income Sweep Registration Code

Required. The registration code for the KEMID income sweep investment security holdings. Enter the appropriate code or use the lookup  to find it.

Income Sweep Registration Code Description

Display only. Your institution's description for the income sweep registration code.

 

Sweep Principal Cash Limit

Required. The least amount of uninvested principal cash that your institution wants to hold in the KEMID.

Principal Sweep Investment

Required. The security ID of the security that is to be used for the sweep investment of principal cash. Enter the appropriate security ID or use the lookup  to find it.

Principal Sweep Investment Description

Display only. Your institution's description of the principal sweep investment.

 

Principal Sweep Registration Code

Required. The registration code for the KEMID principal sweep investment security holdings.

Principal Sweep Registration Code Description

Display only. Your institution's description of the principal sweep registration code.

 

Cash Sweep Frequency

Required. A code identifying the regularly scheduled date on which activity for this model is to take place. Enter the appropriate code or use the lookup  to find it.

Cash Sweep Frequency Description

Display only. Your institution's description of the cash sweep frequency.

 

Cash Sweep Next Due Date

The date the next cash sweep is to occur. Initially filled in by the system based on the frequency code. Change the date by entering another date or selecting it from the calendar .

Active Indicator

 

Indicates whether this cash sweep model is active or inactive. Remove the check mark to deactivate this cash sweep model.
