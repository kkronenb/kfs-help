Security Tab

The Security tab identifies specific securities (if any) to be used as selection criteria when calculating fees.

     This tab is available for your use only if the Fee Type value is ‘B’ (balance) in the Fee Method tab.

     If the Select by Security box is checked in the Fee Method tab, the Security tab must contain at least one security for which the Include box is checked.

Security tab definition

Title

Description

Security ID

Required if security ID is to be used as a criterion when calculating fees; otherwise, not used. Specifies a security ID to be used. Each security ID uniquely identifies an asset or liability.

Security ID Description

The description of, or name for, the security.

Include

Indicates whether this security ID is to be used as a basis for fee calculation. Uncheck the box to discontinue using this security ID for the fee calculations for this fee method.
