Document Layout

The Report Earn Paygroup document includes the Edit Report Earn Paygroup tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

The report type code associated with an earn code and paygroup combination.

The earn code value you want to associate with this report type code.

Paygroup

The paygroup value associated with this report type code.

Active

If checked this row will be considered when effort certification reports are created. If unchecked this row will be ignored.
