Shared Tables with Other Modules

Since Budget Construction combines base budget data from the General Ledger and position and incumbent data from your institution’s payroll system, it shares tables with the Labor Distribution module. Additionally, some Budget Construction functionality is controlled by the Function Control table, which is shared by the Budget Adjustment document within the Financial Processing module.

     Fiscal Year Function Control Table: Some function control codes play an important role in activating and deactivating certain parts of Budget Construction functionality and in starting and stopping the budget construction cycle.
BCACTV:  Activates the BC Module
BCUPDT:  Activates the edit mode within the BC Module
BSSYNC:  Activates batch synchronization with payroll
PSSYNC:  Activates online synchronization with payroll
CSUPDT:  Allows CSF Update
BCGENE:  Indicates that the Genesis job is currently running

   For information about these function control codes, see Function Control Code in the Guide to the KFS Maintenance Menu.
