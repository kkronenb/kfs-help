Document Layout

When you click Separate from the Actions column in the Asset Lookup or Asset Payment Lookup results, the system displays the Asset Global document. The Asset Global document contains the Asset Acquisition Type tab, the Asset Information tab, the Asset Detail Information tab, Location tab, the Add Payments tab, and the Recalculate Total Amount tab in addition to several standard e-doc tabs.

    For more information about the standard tabs, see Standard Tabs.
