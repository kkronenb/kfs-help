Letter of Credit Fund Group document

The LOC document includes the Edit Letter of Credit Fund Group tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Edit Letter of Credit Fund Group tab definition

Title

 

Description

Letter Of Credit Fund Group Code

A code to represent an abbreviation of the letter of credit (loc) fund group description. Codes exist for most large federal departments and agencies as these agencies use LOC funding for their grants and cooperative agreements. This code is used to aggregate expenditures on all awards from a particular loc funding group in order to determine how much cash needs to be requested from the agency. Up to four characters.

Letter Of Credit Fund Group Description

Required. A text description of the LOC fund group in order to assist in determining which LOC fund group code to assign to a given award.

Active Indicator

 

Indicates whether this letter of credit fund group code is active or inactive. Remove the check mark to deactivate this code.
