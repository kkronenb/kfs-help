Organizations Tab

The Organizations tab indicates which organizations at your institution are involved in this project. Multiple organizations can be added but only one can be marked as the 'Primary' organization.

Organizations tab definition

Title

 

Description

Chart Code

 

Required. Enter the code to uniquely identify the Chart of Accounts an organization belongs to, or search for it from the Chart lookup .

Organization Code

 

Required. Enter the code to uniquely identify the campus, department and school associated with this proposal, or search for it from the Organization lookup .

Name

Display-only. The system-generated name of the organization represented by the organization code.

Primary

Optional. Check this box to indicate that this is the primary organization sponsoring the proposal. There must be one (and only one) primary organization. Defaults to checked for new proposals.

Click  to add the new organization to the document.
