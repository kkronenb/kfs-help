Numerous reports are available for the KFS Effort Certification module.

Effort Certification submenu group

Title

 

Description

Duplicate Certifications Report

Identifies duplicate effort certification reports generated for a given fiscal year and report number.

Effort Certification Extract Build

Displays the effort distribution lines generated by the Effort Certification Extract Build process for a given fiscal year, employee and report number.

Outstanding Certifications by Chart/Org Report

For a given chart and organization, displays the number of effort certification reports that have not been fully approved.

Outstanding Certifications By Report

Indicates the number of effort certification reports of a given report number that have not been fully approved.

These reports are primarily intended to assist central office users in the creation and tracking of effort reports.
