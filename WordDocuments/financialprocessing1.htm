Financial Processing

•      If you create a new Disbursement Voucher Payment Reason code for Vendor Payments that are initiated by clicking the Vendor Payment button on the TEM documents, that code will need to be added to VALID_PAYEE_TYPES_BY_PAYMENT_REASON and VALID_OBJECT_LEVELS_BY_PAYMENT_REASON.

    VENDOR_PAYMENT_REASON_CODE will also need to be updated with this value.
