Creating a Profile for an Employee

1.     Select TEM Profile from the Travel submenu group in the Lookup and Maintenance submenu on the Main Menu tab.

2.     Log into the KFS as necessary.

3.     The TEM Profile Lookup will open.

4.     Click the  button in the upper right corner of the lookup screen.

5.     The system displays the TEM Profile From Kim Person Lookup screen.

6.     Search for the traveler’s Person record.

7.     In the search results table, click the create new profile link for the appropriate record. The system displays the TEM Profile document including pertinent data from the traveler’s Person record.

8.     Enter additional data as needed for this TEM Profile and click .
