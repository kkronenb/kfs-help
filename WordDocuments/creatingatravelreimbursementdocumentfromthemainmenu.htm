Creating a Travel Reimbursement Document from the Main Menu

To create a reimbursement request from the Main Menu:

6.     Select Travel Reimbursement from the Travel subsection of the Transactions submenu on the Main Menu tab.

7.     Add information to accurately report activities and expenses for the trip.

8.     If there is more than one object code in the Summary by Object Code tab, use the Assign Accounts tab to create the accounting lines.

9.     Enter accounts and percentages or amounts in the Assign Accounts tab, click  to add each accounting line.

10.  Then click  to update the Accounting Lines tab with the object codes in Summary by Object Code tab and the accounts and amounts in the Assign Accounts tab.

11.  If there is only one object code in the Summary by Object Code, the object code and amount to be distributed will display in the Accounting Lines tab. Enter the account and click . Add additional accounts and amounts as desired.

12.  Select a due date and payment method.

13.  Click to send the Travel Reimbursement into Workflow.
