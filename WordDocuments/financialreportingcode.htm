Financial Reporting Code

 >  >  >  >

The Financial Reporting Code document is used to define this optional attribute of sub-account that can be used for reporting. Financial reporting codes can roll up to other financial reporting codes creating a hierarchy, similar to the way that organizations can map to other organizations. Decision support queries may be written to leverage information by financial reporting code hierarchy.
