Travel, Entertainment and Moving > Attribute Maintenance Documents > Traveler Type

Document Layout

 

The Traveler Type document includes the Edit Traveler Type tab. The system automatically enters data into both the Old and New sections in this tab. Selected data fields are available for editing.

Edit Traveler Type tab definition

Title

Description

Traveler Type

Display Only. The code for this type of traveler.

Traveler Type Name

Required. The descriptive name assigned to this type of traveler.

Advances

Indicates whether this type of traveler can request an advance (checked) or not (unchecked).

    This functionality has not been developed.

Active

Indicates whether this Traveler Type record is active (Y) or inactive (N).
