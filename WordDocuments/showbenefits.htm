Show Benefits

The show benefits button causes the system to display the Single Line Benefit Impact screen. This screen shows the calculated benefit amounts for each fringe benefit type. Fringe benefits rates are defined in the Labor Distribution module.

   For information about fringe benefit rates, see Labor Benefits Calculation.

To use this function, click the  button located on the budget line.

 

The system displays the Single Line Benefits Impact screen.
